Introduction

Various Steps to Train an ML Model on some data

We have classified the various steps of the training process into 5 categories:

  1. Data Processing
  2. Missing Value Analysis
  3. Encoding Techniques
  4. Training Models
  5. Testing Trained Models

Let us see the purpose and tasks of each one in detail:

Data Processing:

Here, we extract those features which can be extracted without performing any single change to a dataset. Therefore it will be valid to every dataset.

Missing Value Analysis:

In missing value analysis, we will be detecting outliers if any and in that case it will replace missing values by median. If outliers are not there mean will be used and for categorical features mode will be used to fill the missing value. There are three main types of missing data: (1) Missing Completely at Random (MCAR), (2) Missing at Random (MAR), and (3) Missing Not at Random (MNAR).

Encoding Techniques:

Encoding Techniques will include replacing Categorical Features by numbers. Suppose a column has features red, blue, green only so we can replace them by 0, 1, 2 respectively.

Training Models:

In this step we will be training our model on the data. We will be using various algorithms to train our model.

Testing Trained Models:

In this step we will be testing our trained model on the data. We will be using various algorithms to test our model. In this we have to make a graphical tool, in which user can generate a graph just by selecting any 2 features, from data set and can make different kinds of plots like scatter, box, violin etc.

We will also be selecting the best 2 features to select for training the model.
